# Linux
### _Знакомство с консолью Linux_
> **Цель**:\
>Освоение базовых команд консоли Linux.

##### Установить виртуальную машину VirtualBox, установить с нуля или запустить скаченный образ Ubuntu После login-а в Linux в консоле выполните следующие действия:
###### 1.Узнайте полный путь текущего каталога:
```console
pwd
```
###### 2. Выведите на экран ПОЛНОЕ содержимое текущего каталога:
```sh
ls -lah
```
###### 3. Cоздайте каталог otusHW:
```sh
mkdir otusHW
```

###### 4. перейдите в каталог otusHW:
```sh
cd otusHW
```

###### 5. Выведите в консоль информацию о файловой системе (точки монтирования, занимаемый размер, свободное место):
```sh
df -h
```
###### 6. Информацию из пункта 5 выведите в файл fsInfo:
```sh
df -h > fsInfo
```

###### 7. Выведите на экран содержимое файла fsInfo:
```sh
cat fsInfo
```

###### 8. Выведите на экран строчки файла fsInfo, в которых есть цифра 1 или цифра 2:
```sh
cat fsInfo | grep [1-2]
```

###### 9. Cделайте копию файла fsInfo - fsInfoСopy:
```sh
cp fsInfo fsInfoCopy
```

###### 10. Cделайте копию файла fsInfo - fsInfoСopy2:
```sh
cp fsInfo fsInfoCopy2
```

###### 11. Переименуйте fsInfoСopy2 в fsInfoСopy3:
```sh
mv fsInfoCopy2 fsInfoCopy3
```
###### 12. Удалите fsInfoСopy3:
```sh
rm fsInfoCopy3
```

###### 13. Выведите на экран первые строки fsInfoСopy:
```sh
head fsInfoCopy -v -n 5
```

###### 14. Выведите на экран последние строки fsInfoСopy:
```sh
tail fsInfoCopy -v -n 5
```

###### 15. Cоздайте файл runIt:
```sh
touch runIt
```

###### 16. В vi откройте файл runIt и добавьте команду копирования fsInfo в fsInfoFinal:
```sh
vi runIt
cp fsInfo fsInfoFinal
```

###### 17. Cделайте runIt исполняемым:
```sh
chmod a+x runIt
```

###### 18. Выполните runIt и убедитесь, что он отработал как надо работа сделана:
```sh
./runIt
```
